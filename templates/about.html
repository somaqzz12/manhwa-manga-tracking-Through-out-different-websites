Help / How it works

Manga Watchlist is a discovery and tracking app. It is not a manga reader and does not host chapters or panel images.

Getting started

Search by title: Use Discover to search manga/manhwa names and compare available sources.

Paste any URL: You can paste a public source URL. If supported, metadata is resolved automatically; otherwise you can still track it manually.

Add to library: Use the Add button from cards, or go to Add URL.

Open original site: External links take you to third-party source sites in a new tab.

Support labels

Automatic — official APIs/feeds with stable metadata resolution.

Supported — site-specific adapters with chapter parsing.

Experimental — generic page detection; quality varies by site layout.

Manual — URL + progress tracking only.

Unavailable — blocked or unsupported for automated access.

Browser extension

The extension helps capture page metadata from supported domains and send it to your library. If no verified web store listing is available, install it via manual ZIP/unpacked flow on the Extension page.

Safety and responsibilities

We do not host manga chapters/pages. Users are responsible for URLs they add. Source links may point to third-party websites not controlled by Manga Watchlist. Prefer official and licensed sources where available.
