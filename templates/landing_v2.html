How it works

  1. Search or paste — titles and URLs both land in the same discovery flow.
  2. Compare — see which sources have fresher chapters before you commit.
  3. Track — add to your library; automatic checks run for supported sites, manual for the rest.
  4. Request — unsupported domains go to Source Requests for future adapters.

Supported sources

Automatic

MangaDex, MANGA Plus, ComicWalker, and other official/API-backed catalogs.

Supported adapters

Popular reader sites with dedicated parsers — see the live Sources page for status.

Experimental & manual

Generic detectors plus manual bookmarks — every URL is accepted.

Extension companion

Optional Chrome extension: detects chapter pages and syncs progress to your signed-in library. It never replaces the website — discovery, comparison, and library management stay here.

Safety

We do not host chapters or panel images. Manga Watchlist stores metadata, URLs, and your reading progress only — never a manga reader or image cache.
