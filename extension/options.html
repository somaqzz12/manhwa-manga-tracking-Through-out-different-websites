Manga Watchlist — Settings

By default the companion talks to the official Manga Watchlist server. Override the URL only if you run your own deployment.

Backend URL

The Manga Watchlist site this extension syncs to. Leave the default to use the official server; only change it if you're hosting your own copy. The popup pings /healthz here and the badge polls /api/unread-count. Saving a custom URL will ask Chrome for host permission to that origin (the pre-shipped backend domains are already allowed).

Tracking behavior

Auto-track silently records progress without showing the prompt. Cooldown controls how long "Not now" stays snoozed.

Local data

Wipes the extension's locally stored settings, debug log, and snooze records. Does not touch your account or library on the server.

Debug log

The most recent 25 events the extension recorded. Useful when something doesn't sync.
